# Architektur

## Ziel

Das Repo überführt lokales Material der Erzieherausbildung in eine prüfbare Wissensstruktur und daraus abgeleitete Darstellungen. Es zeigt sowohl den dauerhaften Ausbildungskanon als auch freigegebene aktuelle Halbjahresarbeit, ohne beide Autoritätsebenen zu vermischen.

## Quellenkorrektur

Die gültige lokale Quelle ist `~/iCloud/Drive/inbox/erzieherausbildung`. Die früher verwendeten Ordner `1. Semester` und `2. Semester` sind ausgeschlossen.

## Ebenen

1. **Lokale Quellenebene:** Roh-PDFs und vollständige Arbeiten bleiben lokal.
2. **Kanonische Wissensebene:** Lernkarte, Details, Exzerpte, Theorien und Wissensnetz bilden den dauerhaft geprüften Stand.
3. **Aktuelle Arbeit:** anonymisierte, strukturierte Ergebnisse eines konkreten Halbjahres mit eigenem Lebenszyklus.
4. **Kristallisierung:** geprüfte Entscheidungen verbinden aktuelle Arbeit mit konkret geänderten Kanonzielen oder archivieren sie.
5. **Darstellungsflächen:** Web, Obsidian Canvas und spätere Schauwerk/Miro-Flächen bleiben abgeleitet.

## Autoritätsfluss

```text
lokale Originale
      │
      ├──► Exzerpte / Details / Theorien ──► Kanon
      │
      └──► aktuelle Arbeit ──► Prüfung ──► Kristallisierung ──► Kanonänderung
                                      │
                                      └──► Archiv / Verwerfung

Kanon + aktuelle Arbeit ──► Web-Lesefläche
Kanon                    ──► Canvas-Denkfläche
```

## Alternative Sinnachse

Neben `Lernfeld → Quelle` bleibt die handlungsbezogene Achse wichtig:

- professionelle Rolle klären;
- Beziehung, Eingewöhnung und Aufsicht gestalten;
- ressourcenorientiert beobachten;
- Bildungsangebote planen;
- mit Eltern kooperieren;
- Querschnittsthemen einordnen.

Aktuelle Arbeit wird an diese vorhandenen Themen angebunden. Das System vermeidet eine zweite chronologische Dokumentenablage.

## Grenzen

- Die Website ist keine Eingabe- oder Wahrheitsinstanz.
- Aktuelle Arbeit wird nicht automatisch kanonisch.
- Vollständige Schülerarbeiten und personenbezogene Falldaten werden nicht veröffentlicht.
- Ein leerer aktueller Bestand ist ein gültiger Zustand.
